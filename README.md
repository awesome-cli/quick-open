## About

Extended `open` command with default dir set to current.

## Prerequisites

- Node.js
- npm/Yarn

## How to Install

First, install the CLI by npm:

```sh
$ npm install -g quick-open
```

Or Yarn:

```sh
$ yarn global add quick-open
```

## How to Use

<!-- TODO update params -> [] or <> -->
```sh
$ quick-open [options] [dir]
```
